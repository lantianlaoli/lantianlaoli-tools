# Lantian Tools Docs

This folder is organized by product feature and shared vendor/API dependency.

## Feature Docs

- `features/bulk-clone/` - Bulk photo clone workflow, workbook formats, and implementation notes.
- `features/ecommerce-assets/` - Ecommerce image and video asset generation workflow.
- `features/expo-company-atlas/` - Expo company atlas workflow and source assets.
- `features/reviewzon/` - Reviewzon PRD, design notes, validation screenshots, and sample workbook.
- `features/shenzhen-expo-hunter/` - Shenzhen Expo Hunter workflow and research notes.

## Vendor/API Docs

- `vendors/kie/` - KIE image, video, upload, and media API references.
- `vendors/rapidapi/reddit/` - RapidAPI Reddit endpoint references.
- `vendors/steady/` - Steady API reference.

New feature-specific docs should go under `features/<feature-name>/`. Shared third-party API references should go under `vendors/<vendor-name>/`.
