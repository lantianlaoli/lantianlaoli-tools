# Lantian Tools

Tiny tools for solo founders.

Lantian Tools is a collection of focused workspaces for one-person companies: batch product image generation, ecommerce media production, expo company documentation, and buyer-intent research.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to use the tools.

## Scripts

```bash
pnpm type-check
pnpm lint
pnpm test
```
